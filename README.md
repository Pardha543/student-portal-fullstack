# 🎓 Student Portal – Full Stack (JWT + Role Based Security)

A secure full-stack Student Portal built using **Spring Boot (Backend)** and **React (Frontend)** with JWT authentication and role-based authorization.

---

## 🔥 Features

- ✅ JWT Authentication
- ✅ Role-Based Authorization (ADMIN / STUDENT)
- ✅ Custom JWT Filter
- ✅ Stateless Security
- ✅ Admin Dashboard
- ✅ Create User (Admin Only)
- ✅ View Users (Admin Only)
- ✅ Student Dashboard
- ✅ Protected Routes
- ✅ CORS Configured
- ✅ Modern Dark UI

---

## 🛠 Tech Stack

### Backend

- Java 17
- Spring Boot
- Spring Security
- JWT (JJWT)
- Hibernate / JPA
- MySQL

### Frontend

- React (Vite)
- Axios
- React Router
- Modern CSS Styling

---

## 🔐 Authentication Flow

1. User logs in
2. Backend generates JWT token
3. Token stored in localStorage
4. JWT sent in Authorization header
5. Backend validates token via custom filter
6. Role-based access enforced

mvn spring-boot:run

Backend runs at:
http://localhost:8080

Frontend

npm install
npm run dev
